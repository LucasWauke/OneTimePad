Esse algoritmo foi reproduzido para uma apresentação na faculdade de Segurança da Informação

O que ele faz é apenas transformar os textos em código asc, depois para binário e é feito um xor de cada bit, gerando outro binário que é convertido em decimal. Esse decimal será um código da tabela ASCII que será convertido para caractere, gerando assim, um texto criptografado.

Nota: é um algoritmo primitivo, feito apenas com o intuito de entender como funciona e apresentá-lo em um trabalho acadêmico.
